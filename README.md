# AfriTalent 

> La première plateforme de freelance tech dédiée à l'Afrique.

## Informations

- **Auteur** : Mouhamadou Moustapha Gueye
- **Classe** : Licence 1 Informatique — Groupe ISI
- **Année** : 2025–2026
- **Projet** : Projet Semestre 2

## Description

AfriTalent est un site vitrine fictif d'une plateforme de mise
en relation entre freelances tech africains et entreprises.
Le site présente la plateforme, ses fonctionnalités, ses tarifs,
des profils de freelances, et vise à convaincre les visiteurs
de s'inscrire.

## Technologies utilisées

- HTML5 (structure sémantique)
- CSS3 (Flexbox, Grid, Bento Grid, animations, variables CSS)
- Bootstrap 5 (via CDN)
- Bootstrap Icons (via CDN)
- Google Fonts (Syne + Inter)
- JavaScript Vanilla (DOM, événements, IntersectionObserver, localStorage)
- Git & GitHub Pages

## Pages du site

| Page | Fichier | Description |
|------|---------|-------------|
| Accueil | `index.html` | Hero, Comment ça marche, Catégories, Témoignages, CTA |
| Freelances | `freelances.html` | Catalogue filtrable de 9 profils fictifs |
| Tarifs | `tarifs.html` | 3 plans tarifaires + FAQ accordion |
| À propos | `about.html` | Histoire, Équipe, Valeurs, Chiffres clés |
| Contact | `contact.html` | Formulaire + infos + Google Maps |

## Fonctionnalités JavaScript

-  Animation d'étoiles en arrière-plan
-  Dark Mode / Light Mode avec sauvegarde localStorage
-  Compteurs animés au scroll (IntersectionObserver)
-  Filtrage dynamique des profils freelances
-  Validation complète du formulaire de contact
-  Navbar dynamique au scroll
- ⬆ Bouton retour en haut (smooth scroll)
-  Animations fade-in des sections au scroll
